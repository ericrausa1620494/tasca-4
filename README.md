# tasca-4
separar molecula problema i generar una molecula nova:

l'arxiu h2o.pdb conté les molècules d'aigua del problema en un arxiu separat,

l'arxiu nucleic.pdb conté el DNA del problema en un arxiu separat,

l'arxiu protein.pdb conté la proteina del problema en un arxiu separat,

l'arxiu other.pdb conté el Remdesivir, situat en un arxiu separat,

per últim, els arxius moleculatasca4.pdb i xyd contenen la molècula creada fent servir l'extensió molefacture de VMD.
